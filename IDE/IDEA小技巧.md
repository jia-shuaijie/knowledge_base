## 快速查找接口

双击shift 输入 `/url url地址` 即可找出对应接口
## 快捷键

|         快捷键          |             描述              |
| :------------------: | :-------------------------: |
| ctrl+h \| ctrl+alt+h |       查看某个方法都在哪里被引用.        |
|     Alt+shift+c      |        快速查看项目文件最近的变更        |
|    ctrl+shift+空格     |           实例化预期对象           |
|        ctrl+Q        |       快速查看文档[属性/方法常用]       |
| ctrl+alt+shift+左右箭头  |          方法中参数快速移动          |
|    ctrl+shift+f7     | 文本光标放在`throw`按快捷键后高亮抛出异常的语句 |
|      ctrl+alt+V      |           快速重构变量            |
|                      |                             |
